# 🕶️ 시각장애인을 위한 쇼핑 AI 고도화 계획 (Accessibility Roadmap)

본 문서는 시각장애인 사용자가 온라인 쇼핑 시 겪는 '보이지 않는 불편함(Pain Points)'을 해소하기 위한 심화 기능 및 개발 아이디어를 정리합니다.

## 🌟 핵심 철학: "감각의 번역 (Sensory Translation)"
시각 정보(Image/Text)를 **촉각, 청각, 공간 정보**로 변환하여 전달합니다.
- 단순 정보 전달("빨간색 니트")을 넘어 **경험 전달**("보들보들한 촉감의 톤 다운된 와인색 니트")을 지향합니다.

---

## 🛠️ 기능 구현 아키텍처 (2-Stage Pipeline)

사용자 대기 시간을 최소화하면서 풍부한 정보를 제공하기 위한 전략입니다.

### 1단계: 퀵 가이드 (Fast Track) - **[현재 구현됨]**
- **목표**: 5초 이내 즉각 응답
- **내용**: 제품명, 핵심 특징, 가격, 필수 안전 정보(알레르기/유통기한)
- **형식**: `gpt-4o-mini` 기반의 "친절한 쇼핑 친구" 페르소나 요약

### 2단계: 심층 분석 (Deep Dive) - **[추후 개발]**
- **목표**: 오감 정보 및 상세 데이터 제공 (Background Processing)
- **트리거**: 사용자가 "상세 정보 더보기" 요청 시 또는 백그라운드 로딩
- **내용**: 사이즈표 데이터화, 촉감/제형 묘사, UX 조작성 분석 등

---

## 📂 카테고리별 핵심 분석 포인트 (Key Buying Factors)

### 1. 👕 의류 / 패션 (Fashion)
> "맞는지, 무슨 느낌인지가 제일 중요합니다."
- **사이즈 데이터화**: 복잡한 사이즈표 이미지를 시각장애인이 탐색 가능한 구조적 데이터(JSON)로 변환.
- **촉감(Texture)**: '부드러운', '까슬한', '신축성 있는', '두툼한' 등 텍스트 추출.
- **색상 묘사**: 단순 색상명 대신 톤(Tone)과 느낌 전달 (예: "쨍한 형광 파랑" vs "차분한 네이비").
- **패턴**: 무늬의 크기와 모양 묘사 (예: "손톱 크기의 작은 꽃무늬").

### 2. 🍎 신선식품 (Fresh Food)
> "크기와 손질 상태를 알고 싶어요."
- **크기 비유**: cm/g 단위 대신 친숙한 사물에 비유.
    - 예: "테니스공만 한 사과", "어른 주먹 두 개 크기의 배".
- **손질 여부**: 흙이 묻어있는지, 세척된 상태인지, 껍질째 먹어도 되는지.
- **포장 상태**: 개방형 트레이인지, 밀봉 팩인지 (집을 때 쏟아짐 방지).

### 3. 💄 화장품 / 뷰티 (Beauty)
> "흐르는 제형인지, 어떻게 여는지가 중요해요."
- **제형(Formulation)**: '물 같은 스킨', '꾸덕한 밤(Balm)', '미끌거리는 오일'.
- **용기 타입**: 펌프형, 스포이드형, 단지형(손으로 퍼냄), 튜브형.
- **사용량 가이드**: "완두콩만큼", "500원 동전 크기만큼".

### 4. 🔌 가전 / 디지털 (Electronics)
> "눈 감고 조작할 수 있나요?"
- **조작 인터페이스**: **가장 중요**. 터치식(접근성 낮음) vs 물리 버튼/다이얼(접근성 높음).
- **피드백**: 작동 시 소리(Beep)나 진동이 있는지.
- **포트 위치**: 충전 단자나 이어폰 잭의 위치 설명.

### 5. 💊 영양제 / 의약외품 (Health)
> "삼킬 수 있는 크기인가요?"
- **알약 크기/형태**: 목 넘김 난이도 (예: "손가락 한 마디 크기의 큰 타원형").
- **개봉 안전 장치**: 눌러서 돌리는 안전 캡(Child-lock) 적용 여부.

## 📝 프롬프트 고도화 전략

### "비유(Metaphor) 엔진" 추가
LLM에게 크기나 무게를 설명할 때 **상대적인 비유**를 사용하도록 지시합니다.
> *"크기는 cm뿐만 아니라 '동전', '스마트폰', 'A4용지' 등 주변 사물에 비유해서 설명해 주세요."*

### 데이터 구조화 (JSON Output)
요약문(Summary)과 별도로, 상세 데이터를 기계가 읽기 쉬운 형태로 추출합니다.
```json
{
  "summary": "...",
  "details": {
    "size_table": { ... },     // 의류
    "texture": "부드러움",      // 의류/화장품
    "interface_type": "터치식", // 가전
    "unit_size_desc": "500원 동전 크기" // 식품/알약
  }
}
```
