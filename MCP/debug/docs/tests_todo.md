# 실행 예정 테스트

마지막 업데이트: 2026-02-05

## 높은 우선순위
- 수량 증가 경로: 목표 > 현재; 플러스 클릭만 발생하는지 검증
- 비선택 상품명 수량 변경 (유사 매칭 확인)
- 수량 변경 후 장바구니 추출: 변경된 수량이 추출 결과에 반영되는지 확인

## 중간 우선순위
- 장바구니/검색에서 읽기 전용 요청: 읽기 전용 TTS만 출력되는지 검증
- 복합 의도: "상품 정보 읽고 장바구니에 담아" -> 읽기 TTS 후 명령 실행
- 인터럽트 처리: TTS 중단 시 이후 TTS/명령 전달이 중지되는지 검증

## 낮은 우선순위
- 페이지 이동만 수행: 빠른 ACK + 추가 TTS 없이 올바른 도구 명령만 실행
- 오류 폴백: 추출 실패 시 안전한 폴백 메시지 출력 확인
