# 완료된 테스트

마지막 업데이트: 2026-02-05

## 완료 항목
- 스크립트 입력 흐름 (ASR 미사용): 검색 -> 상품 선택 -> 상품 정보 -> 장바구니 담기 -> 장바구니 추출
- 상품 정보 읽기 파이프라인: 상품 상세 대기 + 지연 읽기 정상 동작 확인
- 장바구니 담기 흐름: 이동 -> 장바구니 추출 -> 확인 TTS 검증
- 장바구니 수량 변경: last_added_product 기반 규칙 타겟 지정; +/- 클릭 생성 (fill/press 미사용)

## 근거
- AI 서버 로그: "Set last_added_product" 이후 장바구니 수량 변경 TTS 및 클릭 명령 확인
- MCP 로그: "click: decrease quantity" + 셀렉터 `...quantity-input .twc-bg-minus-icon` 확인

## 비고
- 수량 변경은 추출 시점의 장바구니 상태에 따라 달라짐. 테스트 당시 현재 수량 4, 목표 수량 2로 마이너스 클릭 2회가 정상 동작.
