# Frontend 개발 할 일 목록 (서해령 담당 - 리팩터링 제외)

## 에픽: 인증

### Story: Feat: 일반형 아이디 찾기 화면 구현
- **Role**: FE
- **담당자**: 서해령
- **설명**: [FE] 사용자는 분실한 아이디를 찾기 위해 아이디 찾기 화면을 사용하고 싶다.
- **Story Point**: 3 SP
- **완료 조건**:
  - [ ] 이메일 입력 폼 UI 구현
  - [ ] 인증번호 전송 및 확인 UI 구현
  - [ ] 아이디 찾기 결과 팝업/화면 구현

### Story: Feat: 일반형 비밀번호 찾기 화면 구현
- **Role**: FE
- **담당자**: 서해령
- **설명**: [FE] 사용자는 분실한 비밀번호를 재설정하기 위해 비밀번호 찾기 화면을 사용하고 싶다.
- **Story Point**: 3 SP
- **완료 조건**:
  - [ ] 이메일 인증 UI 구현 (아이디 찾기와 유사)
  - [ ] 새 비밀번호 입력 및 확인 폼 구현
  - [ ] 비밀번호 변경 완료 후 로그인 페이지 이동

### Story: Feat: 인증 관련 API 연동 및 예외 처리
- **Role**: FE
- **담당자**: 서해령
- **설명**: [FE] 사용자는 원활한 인증 과정을 위해 API와 연동된 기능을 사용하고 싶다.
- **Story Point**: 3 SP
- **완료 조건**:
  - [ ] `POST /auth/findIdByEmail` 연동
  - [ ] `POST /auth/resetPassword` 연동
  - [ ] 이메일 인증번호 발송/확인 API (`emailService` 또는 백엔드 API) 연동 확인
  - [ ] 500/400 에러 처리 및 사용자 알림
